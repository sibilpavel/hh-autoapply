# 🚀 Auto Apply Extension

Браузерное расширение для автоматизации откликов на вакансии с поддержкой кастомного сопроводительного письма.

---

## ✨ Возможности

- Автоматический запуск процесса откликов
- Редактируемое сопроводительное письмо
- Сохранение текста между сессиями
- Дефолтное письмо при первом запуске
- Удобный UI с вкладками (Bootstrap)
- Быстрый старт одной кнопкой

---

## 📦 Установка

1. Склонируйте репозиторий:
```bash
git clone https://github.com/your-username/auto-apply-extension.git
```

2. Откройте Chrome:
```
chrome://extensions/
```

3. Включите режим разработчика

4. Нажмите:
   Load unpacked / Загрузить распакованное расширение

5. Выберите папку проекта

---

## 🧩 Как добавить расширение в браузер (Chrome / Edge)

### 1. Открой страницу расширений
chrome://extensions/

### 2. Включи режим разработчика
В правом верхнем углу включи переключатель Developer mode

### 3. Загрузить расширение
Нажми Load unpacked / Загрузить распакованное расширение

### 4. Выбери папку проекта
Укажи папку, где находится manifest.json

### 5. Готово

---

## 🔄 Обновление расширения

1. Перейди в chrome://extensions/
2. Найди расширение
3. Нажми Reload / Обновить

---

## 🧠 Как работает сопроводительное письмо

- При первом запуске используется дефолтный текст:
  Работал со всеми технологиями из вашей вакансии, когда удобно будет созвониться?

- Если пользователь изменяет текст — он сохраняется
- Данные хранятся в chrome.storage.local
- Content script всегда использует актуальное значение
